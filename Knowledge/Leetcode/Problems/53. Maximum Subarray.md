# 53. Maximum Subarray

## Problem
Given an integer array `nums`, find the subarray with the largest sum, and return its sum.

### Examples
- Input: [-2,1,-3,4,-1,2,1,-5,4], Output: 6  
- Input: [1], Output: 1  
- Input: [5,4,-1,7,8], Output: 23  

### Constraints
- 1 <= nums.length <= 10^5  
- -10^4 <= nums[i] <= 10^4
